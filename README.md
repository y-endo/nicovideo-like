# ニコニコ動画風YouTubeプレイヤー
http://nicovideo-like.firebaseapp.com/
YouTubeの動画に対して、ニコニコ動画風のコメントを流すYouTubeプレイヤーなWebサイト。  

React + reduxの学習用に作成。  
コメントデータはFirebaseのFirestoreで管理。
